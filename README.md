# resumo-do-lab
1. A Conta Gratuita de 30 dias (O "Trial" com Crédito)
   sso é o que a maioria das pessoas conhece como a "conta gratuita". Ela oferece duas coisas principais quando você se cadastra pela primeira vez:
  - Crédito de 30 dias: Você ganha um crédito (geralmente o equivalente a $200 USD, que dá uns R$ 1.000) para gastar em qualquer serviço pago do Azure (como máquinas virtuais grandes, bancos de dados robustos, etc.) durante os primeiros 30 dias.12 Meses de
  - Serviços Populares: Após os 30 dias (ou enquanto o crédito dura), você ainda tem acesso a versões limitadas de serviços populares (como VMs pequenas, Banco de Dados SQL, etc.) gratuitamente por 12 meses.
    
2. O Nível Gratuito do Azure DevOps (Onde o Pipelines mora)
   Aqui está a parte mais importante para você: O Azure Pipelines não depende daquele crédito de 30 dias.
   O Azure Pipelines faz parte do "Azure DevOps Services", que tem seu próprio nível gratuito "Sempre Gratuito" (Always Free):
  - Para Projetos Privados (Seu caso):
       - 1 Job Paralelo Gratuito: Você pode rodar um pipeline de cada vez.
       - 1.800 minutos (30 horas) por mês: Você tem 30 horas de tempo de máquina (agentes de build hospedados pela Microsoft)              por mês para rodar seus builds e deploys.
  - Para Projetos Públicos (Open Source):
        - É ainda mais generoso, com 10 jobs paralelos gratuitos e muito mais tempo.
